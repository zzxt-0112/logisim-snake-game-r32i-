# -Logisim-R32i-CPU-
基于Logisim的R32i指令集流水线CPU仿真下贪吃蛇游戏中的汇编代码

##硬件接口
硬件输入采用LW，输出采用SW，使用虚拟地址

W:0x40000014,S:0x40000018,A:0x4000001C,D:0x40000020,清空寄存器:0x40000024,绝对PC:0x40000028

WASD读入为0/1，所以需要将键盘输入ASCII码先硬件转码，比较器即可

32*32VGA:0x30000000~0x30040000，我知道多了，但因为直接复制的256*256VGA的电路所以是这个数值

七段数码管:0x40000000，话说也没有谁会再搓一个二进制转十进制显示了吧(大概吧

##一个可能的问题
为什么初始化时会有一大段人工递增而不是直接循环,因为是本人仿真为流水线CPU,但是没做分支预测T_T所以为了优化性能尽量降低跳转次数,从sleep块循环次数也能看出来影响有多大
